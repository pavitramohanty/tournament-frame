# 🏐 Volleyball Tournament 2026 — Photo Booth

A mobile-first React web app that lets tournament participants take a photo using their phone camera and automatically composite it inside the official Volleyball Tournament 2026 frame. Users can retake or download their photo instantly — no app install required.

---

## ✨ Features

- 📸 Live camera feed with tournament frame overlay
- 🔄 Front / back camera toggle
- 🖼️ Auto-composites user photo inside the frame
- ⬇️ One-tap download of the final photo
- 🔁 Retake option before saving
- 🎉 Thank you message after download
- 📱 Mobile-first, works on iOS Safari & Android Chrome
- ⚡ Fully client-side — no backend, no login

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| React 18 + Vite | UI framework and build tool |
| react-webcam | Camera access |
| react-icons | Button icons |
| HTML5 Canvas API | Photo compositing |
| GitHub Pages | Hosting |
